RAMKLEENER — MASTER CONTEXT & META v2.1

A pip-installable Python CLI tool that scans system processes across Windows/Mac/Linux,
identifies memory-heavy background tasks, and terminates them safely using a tiered protection model.

ONE-LINER
"Scans system processes, filters using a strict safety model, and terminates known
memory-heavy background tasks with user confirmation — directly from the terminal."

SAFETY MODEL
PROTECTED       — hardcoded, locked forever, OS + critical services
user can extend via user_protected in config

SAFE_TO_KILL    — known non-critical background processes
user can extend via user_kill_list in config

UNKNOWN         — always skipped (safe by default)

CONFIG — ~/.ramkleener/config.json
{
"user_protected": [],
"user_kill_list": [],
"threshold_mb": 50
}

MENU

1. Scan            — show flagged processes + RAM usage
2. Kill all        — terminate all flagged processes with summary
3. Kill one by one — grouped flow (y=kill n=skip q=quit)
4. Help            — usage explanation
5. About           — version, author, GitHub
6. Exit

KEY RULES

* never terminate unknown processes
* scan → display → confirm → execute (strict order)
* threshold filtering is mandatory (min 10 MB, default 50 MB)
* errors must never crash the program
* cleaner executes, never decides
* PROTECTED overrides all conflicts (no config override possible)
